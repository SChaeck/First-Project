[Github에 처음 코드 업로드]
git init : 초기화
git add . : 추가 파일
git add FileName : FileName 파일만 추가 (권장)
git status : 상태확인 (선택)
git commit -m "first commit" : 히스토리 만들기 ("" 내부가 이름)
git remote add origin https://github.com/SChaeck/First-Project.git : Github와 로컬 프로젝트 연결
git remote -v : 잘 연결됐는지 확인 (선택)
git push origin master : Github로 올리기 (master 자리에는 branch 이름이 들어가면 됨.)
